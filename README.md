# student-enrollment-sql-analysis
SQL analysis of a student enrollment database using MySQL.

This project demonstrates SQL skills by analyzing a university student enrollment database.

The project answers common business questions using MySQL.

# Database Tables
- Student Information
- Courses
- Enrollments

# SQL Skill used
- SELECT
- WHERE
- INNER JOIN
- LEFT JOIN
- GROUP BY
- HAVING
- ORDER BY
- Aggregate Functions
- Subqueries

# Business Questions Anawered
✔ Retrieve all students

✔ List students enrolled in each course

✔ Find students with a GPA above 3.5

✔ Count students by department

✔ Calculate average GPA by department

✔ Find the most popular course

✔ Find students not enrolled in any course

✔ Display students who earned an A grade

✔ Show total credits taken by each student

✔ Rank students by GPA
